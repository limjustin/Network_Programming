# Understanding Protocol

### **Protocol**

- 데이터를 주고 받는 상호 간에 미리 약속된 규약
- 주고 받는 데이터의 구조, 해당 구조 속에 포함된 각 부분의 의미, 데이터를 주고 받는 시간, 절차, 속도 등을 포함
